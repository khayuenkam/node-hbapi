Unofficial Hypebeast.com API
============================
This is a side project for me to learn Nodejs and web scrapping.
Feel free to fork and contribute!

How To Start
=============
1. git clone this repo
2. Install and start redis
3. npm install in the repo
4. node server.js
5. Enjoy the api
